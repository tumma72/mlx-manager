# index.ts (ui)

## Purpose

Barrel export for reusable UI primitive components. Exports base components used throughout the application for consistent styling and behavior.

## Exports

- `Button` - Button component with variants and sizes
- `Card` - Card container component
- `ConfirmDialog` - Confirmation dialog modal
- `Input` - Form input component
- `Select` - Form select/dropdown component
- `Badge` - Badge/tag component
- `Markdown` - Markdown renderer component
- `ThinkingBubble` - Loading indicator bubble

## Dependencies

None (imports Svelte components)
