# index.ts (components)

## Purpose

Root barrel export for all UI components. Re-exports all components from the ui, layout, servers, models, and profiles subdirectories for convenient access.

## Exports

- All exports from `./ui` (Button, Card, Input, etc.)
- All exports from `./layout` (Navbar)
- All exports from `./servers` (ServerTile, ProfileSelector, etc.)
- All exports from `./models` (ModelCard, FilterModal, etc.)
- All exports from `./profiles` (ProfileForm, ProfileCard)
