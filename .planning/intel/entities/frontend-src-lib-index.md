# index.ts (lib)

## Purpose

Main library exports for the SvelteKit frontend. Re-exports the API layer, stores, and utilities for convenient access throughout the application. This is the primary import point for shared functionality.

## Exports

- All exports from `./api` (types, client, ApiError)
- All exports from `./stores` (authStore, serverStore, etc.)
- All exports from `./utils` (formatBytes, cn, reconcileArray, etc.)
