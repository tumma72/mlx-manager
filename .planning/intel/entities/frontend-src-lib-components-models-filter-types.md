# filter-types.ts

## Purpose

Defines filter state and constants for model filtering on the models page. Provides the FilterState interface and pre-defined options for architecture and quantization filters.

## Exports

- `FilterState` - Interface with architectures, multimodal, and quantization arrays
- `ARCHITECTURE_OPTIONS` - Array of architecture names for filter dropdown
- `QUANTIZATION_OPTIONS` - Array of bit depths for filter dropdown
- `createEmptyFilters() -> FilterState` - Factory function for empty filter state
